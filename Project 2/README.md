# Backend API Development

## Project Overview
This project is a simple REST API built using Node.js and Express.js. It demonstrates the fundamentals of backend development by creating API endpoints, handling requests, validating user input, and returning JSON responses.

## Features
- GET API endpoint
- POST API endpoint
- JSON response handling
- Basic input validation
- Express.js server
- REST API concepts

## Technologies Used
- Node.js
- Express.js
- JavaScript
- REST API
- JSON

## Installation

1. Clone the repository
2. Open the project folder
3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node server.js
```

or

```bash
npm start
```

## API Endpoints

### GET /

Returns a welcome message.

### GET /users

Returns the list of users.

### POST /users

Accepts user data in JSON format and returns a success response.
